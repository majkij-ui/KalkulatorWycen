# Project Context: Kalkulator Wyceny Wideo

## Purpose
A premium, high-end B2B quoting tool for a Polish video production house. It allows for rapid "Crude" (Szybka) estimates and granular "Detailed" (Szczegółowa) breakdowns across four production phases.

## Target Audience
Polish business clients. The UI and all PDF outputs must be in professional Polish.

## Aesthetic & Vibe
- **Style:** Dark-mode, Glassmorphism, "Boutique Agency" feel.
- **UI Elements:** Blurred backgrounds (`backdrop-blur-xl`), subtle borders (`border-white/10`), and smooth Framer Motion transitions.
- **Interactive:** Custom bipolar sliders (Red for discounts, Orange for margins), independent collapsible cards.
